## üìù Description
Simple-Factions is a small factions plugin for Minecraft. With this plugin, you can create a faction and invite your friends to join. Once you have created a faction, it will be displayed in your name, tab list, and chat.

## ‚ú® Features

### Faction System
This system allows you to create, modify, and delete factions, as well as invite or kick players from your faction.

![Faction system image](path/to/faction-system-image.png)

### Chat System
When you are in a faction, you can use the custom chat system. This allows you to communicate with other members of your faction without others seeing it.

![Chat system image](path/to/chat-system-image.png)
